  • Initiation à la bibliothéque JavaFX

Création d'un petit jeu mêlant jeu de carte et tactical-rpg
  • Approfondissement des connaissances sur le langage C++
  • Principale bibliothéque SFML 2.X
  • Initiation aux threads
 Lors de ce projet je voulais créer un jeu en m'inspirant de Hearthstone et de Dofus je cherchais à mixer les jeux de cartes à jouer et de stratégie.
J'ai pu approfondir quelques notions et notamment débuté mon apprentissage de la programmation multi-threads. Comme je visais avant tout un but pédagogique, une fois à l'aise avec les notions que je visais, je l'ai laissé en suspens.
Création d'une TODO List en ligne
  • Initiation au Javascript(Ajax)
  • Approfondissement du PHP
  • Approfondissement du HTML
  • Approfondissement du CSS
  • Approfondissement du SQL (SGBDR = mysql)

Dans ce projet je voulais créer sur un système basée sur des arborescence un système de gestion mutualisé de todo list dans lequel chaque personne pourrait recevoir un code unique pour accèder à une branche du projet si cette branche est mise en partage publique par le créateur de la liste.

J'étais intéréssé par approfondir mes connaisssance web lors de ce projet. bien que non publié j'ai pu aller au bout de ce projet.

Dans ce projets j'ai voulu manipuler android, en scannant des codes barres intérogeant une API externe, y extraire les valeurs nutritionnelles et permettre de comptabiliser le totales de ces dernières de manière journalière, je voulais également y intégrer des suivi d'entrainement sportif. toutefois je me suis arrêter sur la partie nutritionnelles
